TPCS RSA
TPCS RSA project by Dong Joo Lee

Description

A project to explore the number theory concepts surrounding RSA

Concepts-and-Implementations

Implemented functions for prime generation, primality testing(fermat/miller-rabin) ,modulo inverse solving(extended GCD), fast modulo exponentiation, and RSA encryption/decryption

Primality Testing:

preliminary check by dividing by small primes sieve of Eratosthenes),followed by fermat test, then finally miller-rabin test

Mod Inverse Solver:

perform extended euclidean GCD algorithm

Fast Mod Exponentiation:

left to right/right to left binary method

etc...
